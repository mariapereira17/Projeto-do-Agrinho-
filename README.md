# 🌱 Agro Forte, Futuro Sustentável

Projeto desenvolvido para o Concurso de Programação promovido pelo SENAR-PR e SEED-PR com o tema:

> “Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”.

---

## 📖 Sobre o Projeto

O objetivo deste projeto é apresentar uma página web moderna e interativa voltada à conscientização sobre sustentabilidade no agronegócio.

O site destaca a importância do equilíbrio entre:

- 🚜 Produção agrícola
- 🌿 Sustentabilidade ambiental
- 💻 Tecnologia e inovação
- 🌎 Preservação do meio ambiente

Além disso, o projeto incentiva estudantes e jovens programadores a utilizarem tecnologia para transformar o futuro do campo.

---

## 🎯 Objetivos

- Incentivar a programação e inovação
- Desenvolver consciência ambiental
- Valorizar o agronegócio sustentável
- Integrar educação, tecnologia e meio ambiente
- Promover soluções inteligentes para o campo

---

## 🖥️ Funcionalidades

✅ Página inicial responsiva  
✅ Design moderno com tema agro sustentável  
✅ Seções informativas sobre o concurso  
✅ Cards interativos  
✅ Layout adaptável para celular e computador  
✅ Efeitos visuais e animações suaves
